# 💳 Bank Fraud Transaction Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing bank transaction data to identify fraudulent transaction patterns and generate meaningful business insights. The dataset was cleaned, transformed, and analyzed using Python before creating an interactive Power BI dashboard for fraud monitoring and risk analysis.

The dashboard enables stakeholders to monitor fraud trends, compare fraudulent and legitimate transactions, and support data-driven decision-making.

---

## 🎯 Objectives

- Analyze fraudulent and legitimate transactions
- Identify fraud trends and risk patterns
- Perform data cleaning and preprocessing
- Build an interactive Power BI dashboard
- Generate business insights for fraud prevention
